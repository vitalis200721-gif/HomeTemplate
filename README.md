Edit `lib/demo-config.ts`:
```ts
export const ACTIVE_DEMO: DemoType = "architecture";
```

Edit `lib/site-config.ts`.


# Obsidian Template (Next.js 14+ App Router)

Premium cinematic dark website template intended for studios:
- Architecture
- Interior design
- Creative agency

Tech:
- Next.js 14+ (App Router only)
- Tailwind CSS
- lucide-react icons
- Framer Motion (used safely)
